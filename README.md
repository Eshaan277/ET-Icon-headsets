# ET Icon Headsets

Fully immersive 3D scroll animation website for ET Icon Headsets.

## Deploy to Vercel

1. Upload this zip to [vercel.com](https://vercel.com)
2. Vercel auto-detects Next.js — click Deploy
3. Done!

## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
